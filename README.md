# dotfiles
My dotfiles

# The tools I'm managing here actively are
* `bspwm` - Window manager
* `sxhkd` - Key bindings
* `qutebrowser` - Web browser
* `feh` - Wallpaper
* `nvim` - Text editor
* `zsh` - My shell
* `sxiv` - Image viewer
* `mpv` - Video player
* `lf` - File manager
* `polybar` - Status bar
* `rofi` - Application launcher
* `compton` - Compositor
* `zathura` - PDF reader
